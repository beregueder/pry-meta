A meta package that includes the following gems:

* awesome_print
* pry
* pry-exception_explorer
* pry-nav
* pry-rescue
* pry-stack_explorer

## Installation

Add this line to your application's Gemfile:

    gem "pry-meta"

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pry-meta
